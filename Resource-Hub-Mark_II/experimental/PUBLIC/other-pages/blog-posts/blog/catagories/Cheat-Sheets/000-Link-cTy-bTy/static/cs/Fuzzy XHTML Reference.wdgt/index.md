When allotting space among elements competing for that space, user agents allot pixel and percentage lengths first, then divide up remaining available space among relative lengths. Each relative length receives a portion of the available space that is proportional to the integer preceding the "\*". The value "\*" is equivalent to "1\*". Thus, if 60 pixels of space are available after the user agent allots pixel and percentage space, and the competing relative lengths are 1\*, 2\*, and 3\*, the 1\* will be allotted 10 pixels, the 2\* will be allotted 20 pixels, and the 3\* will be allotted 30 pixels.
