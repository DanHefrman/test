Text unigrams generator tool What is a text unigrams generator?
With this tool, you can create a list of all unigrams from the given text. A unigram, also known as a monogram or 1-gram (because in general they are called n-grams, and n here is 1), is a single unit of text, such as a word or a letter. By default, this tool generates a column of lowercase words. You can switch it to letter mode that will generate a column of letters and also change the unit separator symbol. This utility automatically removes all basic punctuation symbols from unigrams but if it missed some special characters, you can add them to the list of symbols to be removed. If you want to preserve punctuation, you can turn punctuation removal function off with one click in options. You can also control the output case of unigrams. Textabulous!
