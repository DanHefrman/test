<span class="primary">Text fragment extractor tool</span> <span class="secondary">What is a text fragment extractor?</span>

With this online program, you can extract a smaller text fragment from the given text. You can specify the length of the snippet in the options, as well as indicate the position from which to start the extraction. The starting position can be either a positive number or a negative number. The difference is that for positive positions, the characters are counted from the left side to the right side and for negative positions they are counted from the right side to the left side. The length can also be a positive number or a negative number. In case of a positive length, the fragment is extracted to the right side relative to the starting position but in case of a negative length, the fragment is extracted to the left side relative to the starting position. You can also leave the length field empty and then the snippet will be extracted from the starting position to the end of the text. If you enter the "-" symbol in the length field, then the fragment will be extracted from the beginning of the text to the starting position. If your text consists of several lines, you can extract a snippet from each line. Enable the option "Line-by-line" to extract individual fragments from each line. Textabulous!

This example uses the line-by-line option to extract multiple snippets from a class timetable. It uses a negative start value of -15.
